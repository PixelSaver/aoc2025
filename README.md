# Advent of Code 2025 - PixelSaver

Heyo!!

## Running the scripts
To run anything, use
```zsh
cargo run --bin day01
```
Additionally, you can include the input file like so
```zsh
cargo run --bin day01 -- inputs/day02.txt
```
